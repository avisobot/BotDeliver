❓ Frequently Asked Questions
How do I add an extension to the browser? ▼
Go to Browser Menu → Extensions → Select the ZIP/CRX file and upload it.

📺 Video Guide: How to install an extension
📺 Guide: Add extension on Android
How often do tasks arrive? ▼
New tasks arrive every 40–50 minutes. The bot will keep running as long as there are tasks or until it gets stuck.
Can I use other apps while the bot is running? ▼
Yes! You can run two apps at the same time using Split Screen.
📺 Learn how to use split screen →
What happens if the screen turns off? ▼
If the screen turns off, the bot will stop. Solution:
Go to Phone Settings → Find Screen Timeout → Set it to 30 minutes.
What should I do if there's a problem? ▼
Do a Screen Recording, write a description of the problem, and send it to me.
